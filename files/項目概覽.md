# 📦 Claude Chat Ultimate - 項目概覽

## 🎯 項目簡介

Claude Chat Ultimate 是一個功能完整、界面精美的Claude AI聊天客戶端。它集成了Prompt Caching、多模態輸入、知識庫檢索、對話管理等眾多實用功能,旨在為用戶提供最佳的AI對話體驗,同時最大化降低使用成本。

---

## ✨ 核心特性

### 🎨 用戶體驗
- **現代化界面** - 基於tkinter的簡潔美觀GUI
- **三種主題** - Light、Dark、Monokai可選
- **靈活佈局** - 可調整的面板比例
- **Markdown渲染** - 完美展示格式化內容
- **代碼高亮** - 程序員友好的代碼顯示

### 💬 對話功能
- **多模態輸入** - 文字+圖片混合對話
- **上下文管理** - 智能維護對話歷史
- **對話總結** - AI智能壓縮長對話
- **導出功能** - 支持Markdown/HTML/JSON

### 📚 知識管理
- **本地知識庫** - 簡化的RAG系統
- **多格式支持** - PDF/Word/TXT/MD
- **智能檢索** - 關鍵詞搜索和上下文提取
- **自動關聯** - 對話時自動檢索相關內容

### 💰 成本優化
- **Prompt Caching** - 節省高達90%成本
- **實時統計** - 詳細的token和費用追蹤
- **智能壓縮** - 對話總結減少token消耗
- **透明計費** - 清晰展示每一分錢

### 🌐 高級功能
- **聯網查詢** - 獲取實時信息(需支持)
- **圖片識別** - 視覺問答和圖片分析
- **文檔處理** - PDF/Word文本提取
- **批量處理** - 多文檔綜合分析

## 🏗️ 架構設計

### 主要模組

#### 1. ClaudeChatUltimate (主類)
- 界面管理
- 配置持久化
- API調用協調

#### 2. KnowledgeBase (知識庫)
- 文檔存儲
- 檢索引擎
- 內容管理

#### 3. MarkdownRenderer (渲染器)
- Markdown解析
- 文本格式化
- 代碼高亮

### 數據流

```
用戶輸入
    ↓
[前處理]
    ├─ 知識庫檢索
    ├─ 圖片編碼
    └─ 歷史整合
    ↓
[API調用]
    ├─ System Prompt (緩存)
    ├─ 對話歷史
    └─ 當前輸入
    ↓
[響應處理]
    ├─ Markdown渲染
    ├─ 統計更新
    └─ 界面顯示
    ↓
用戶查看
```

## 📈 性能指標

### 成本節省
- 使用Prompt Caching: **50-90%**
- 使用對話總結: **額外30-50%**
- 優化System Prompt: **長期效果顯著**

### 響應時間
- 基礎對話: **2-5秒**
- 圖片分析: **5-10秒**
- 知識庫檢索: **<1秒**

### 資源占用
- 內存: **50-200MB**
- CPU: **低(除API調用時)
- 磁盤: **<10MB** (不含知識庫)
